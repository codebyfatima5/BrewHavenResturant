# BrewHavenResturant
☕ Brew Haven – Resturant Website

A modern, fully responsive Coffee Shop & Restaurant website built using HTML, Tailwind CSS, and JavaScript.
This project showcases a stylish UI with smooth interactions, dynamic menu filtering, and a premium user experience suitable for real-world café or restaurant businesses.

✨ Features
🍔 Fully responsive design (mobile, tablet, desktop)
☕ Dynamic menu with categories (Coffee, Food, Desserts)
🛒 Add-to-cart interactive buttons with animations
🔥 Smooth scrolling and modern UI effects
🎨 Beautiful gradient-based design with Tailwind CSS
📱 Mobile-friendly navigation menu
⚡ Lightweight and fast loading
🖼️ High-quality food & coffee imagery (Unsplash integration)
🛠️ Tech Stack
HTML5
Tailwind CSS
JavaScript (Vanilla)
Font Awesome Icons
Google Fonts (Poppins)
📸 UI Highlights
Hero section with modern gradient design
Animated product cards with hover effects
Clean reservation section
Professional footer with contact info & social links
